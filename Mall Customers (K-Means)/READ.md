Clustering mall customers data using the unsupervised algorithm of K-Means.

This code includes:

- Cleaning the data
- Exploring the data
- Using the elbow method to deduce the optimal number of clusters
- K-Means to to cluster the data
- A 3-D plot of the clusters and their centriods

Dataset was from kaggle: https://www.kaggle.com/shwetabh123/mall-customers
